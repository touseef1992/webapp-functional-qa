This repository contains QA artifacts for the Demo Web App Functional Testing project.

## Structure
- testplan/ - Test plan and related docs
- testcases.xlsx - Master test case suite (60 test cases)
- bugreports.csv - Exported bug reports
- evidence/ - Screenshots and short videos (not included here)
- reports/ - Execution summary and other reports
- exploratory/ - Exploratory testing notes

## How to use
1. Run the demo app: `python app.py`
2. Open `testcases.xlsx` and execute tests one by one
3. Save screenshots to `evidence/screenshots/`
4. Log bugs to GitHub Issues or import `bugreports.csv` into Jira
5. Update `reports/execution-summary.md` after execution
