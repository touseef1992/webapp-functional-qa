# Web App Functional QA Project  
A complete end-to-end QA documentation and testing project demonstrating functional testing, bug reporting, exploratory testing, and evidence-based validation for a sample web application.

This project showcases real QA workflow skills including:
- Test case design  
- Bug reporting  
- Test planning  
- Manual execution  
- Evidence collection  
- Repo organization  
- GitHub Issues workflow  

---

## Project Overview
This project validates the core functionality of a demo web application that includes:

- User Signup  
- Login  
- Dashboard Access  
- Session Handling  
- Input Validation  
- Security tests (including XSS checks)  
- UI & Usability checks
